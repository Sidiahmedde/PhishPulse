# Dataset Setup

This folder is reserved for datasets used in training.

## EPVME Dataset (GitHub)

Preferred layout after download:

```
dataset/EPVME-Dataset/
```

### Option 1: Git clone (recommended)

```bash
git clone https://github.com/sunknighteric/EPVME-Dataset dataset/EPVME-Dataset
```

### Option 2: Manual download

1. Download the repository ZIP from GitHub.
2. Extract it into `dataset/EPVME-Dataset/`.

### Notes
- The EPVME repo is GPL‑3.0 licensed. If you redistribute the dataset or
  derivatives, you must comply with GPL‑3.0.
- Some upstream sources cited by EPVME have additional constraints.
  Keep usage limited to academic research and avoid redistributing raw emails.
